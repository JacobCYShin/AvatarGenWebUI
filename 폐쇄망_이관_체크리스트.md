### 이관 전 체크

- [ ] `static/tailwind.min.js` 파일 존재
- [ ] `static/tailwind.min.js` 크기: 100KB 이상

## ❌ 트러블슈팅

### 문제 1: UI가 깨짐 (텍스트만 보임)

**증상:**
- 스타일이 전혀 적용되지 않음
- 버튼이 파란색 링크처럼 보임
- 레이아웃이 세로로 나열됨

**원인:** Tailwind CSS 파일 누락

**해결:**
```bash
# 파일 확인
ls static/tailwind.min.js

# 파일이 없으면
# → 인터넷 연결된 환경에서 download_tailwind.py 실행 후 재이관
```

### 문제 2: Tailwind 로드 에러 (404)

**증상:** 
브라우저 Console에 다음 에러:
```
GET http://localhost:8000/static/tailwind.min.js 404 (Not Found)
```

**원인:** 
- 파일이 `static/` 폴더에 없음
- 파일 권한 문제

**해결:**
```bash
# 파일 확인
ls -la static/

# 권한 확인 및 수정
chmod 644 static/tailwind.min.js

# 서버 재시작
```

### 문제 3: CDN 에러 (폐쇄망)

**증상:**
브라우저 Console에 다음 에러:
```
GET https://cdn.tailwindcss.com/ net::ERR_NAME_NOT_RESOLVED
```

**원인:** HTML에 CDN 링크가 남아있음

**해결:**
```bash
# index.html 확인
grep "cdn.tailwindcss.com" templates/index.html

# 있으면 수정:
# <script src="https://cdn.tailwindcss.com"></script>
# →
# <script src="/static/tailwind.min.js"></script>
```

## 📊 시스템 요구사항

### 최소 사양
- **Python**: 3.8 이상
- **메모리**: 2GB RAM
- **디스크**: 500MB (웹 서버만)
- **네트워크**: 로컬호스트 (인터넷 불필요)

### 포트 사용
- **8000**: 웹 UI 서버 (이 애플리케이션)
- **7009**: TTS 서버 (별도)
- **8001**: 비디오 생성 서버 (별도)

### 브라우저 호환성
- Chrome/Edge: 90+
- Firefox: 88+
- Safari: 14+
