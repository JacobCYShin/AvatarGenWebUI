# 🚀 빠른 시작 가이드 (분리형 아키텍처)

## 서버 실행 순서

### 1단계: TTS 서버 실행 (터미널 1)

```bash
cd TTS_server_only
source /database/venv/melotts/bin/activate
uvicorn tts_server:app --host 0.0.0.0 --port 7009
```

**확인**: `http://localhost:7009/` 접속 시 응답 있어야 함

---

### 2단계: 비디오 생성 서버 실행 (터미널 2)

```bash
cd video_generation_server
python video_server.py
```

**확인**: 콘솔에 "✅ 서버 준비 완료!" 메시지 표시

---

### 3단계: 웹 데모 서버 실행 (터미널 3)

```bash
cd web_demo
python app.py
```

**확인**: 콘솔에 서버 시작 메시지 표시

---

### 4단계: 브라우저에서 접속

```
http://localhost:8000
```

---

## 사용법

1. **목소리 모델 선택** (자연스러운 / 아나운서)
2. **텍스트 입력** (예: "안녕하세요. 오늘 날씨가 참 좋네요.")
3. **"비디오 생성하기" 클릭**
4. **진행 상황 확인 및 대기**
5. **완료 후 비디오 재생 및 다운로드**

---

## 포트 요약

| 서버 | 포트 | 용도 |
|------|------|------|
| TTS 서버 | 7009 | 텍스트 → 음성 변환 |
| 비디오 생성 서버 | 8001 | 음성 → 비디오 생성 |
| 웹 데모 서버 | 8000 | 웹 인터페이스 |

---

## 문제 해결

### TTS 서버 연결 실패
```bash
# 확인
curl http://localhost:7009/

# 해결: TTS 서버를 먼저 실행하세요
```

### 비디오 생성 서버 연결 실패
```bash
# 확인
curl http://localhost:8001/health

# 해결: 비디오 생성 서버를 실행하세요
```

### GPU 메모리 부족
→ `video_server.py`에서 `batch_size=8`을 더 작은 값으로 변경

### 모델 파일 없음
→ 아바타 경로 및 체크포인트 경로 확인
