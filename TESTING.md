### 2. 기능 테스트

#### 예시 텍스트 삽입
1. "예시 1" 버튼 클릭
2. 텍스트 입력창에 텍스트가 자동 입력됨
3. 글자 수 카운터가 업데이트됨

#### 상세 설정 토글
1. "상세 설정" 클릭
2. 음성 설정, 프레임 설정, 타이밍 조정 섹션이 표시됨
3. 각 섹션 헤더 클릭 시 내용 펼쳐짐/접힘

#### 출력 타입 변경
1. "오디오만 생성" 선택
2. 아바타 선택이 숨겨짐
3. 버튼 텍스트가 "오디오 생성하기"로 변경
4. 프레임 설정과 타이밍 조정이 숨겨짐

### 3. 브라우저 개발자 도구 확인

**F12** 키를 눌러 개발자 도구 열기

#### Console 탭

예상되는 메시지:
```
(아무 에러 없음)
```

❌ **에러가 있는 경우**:
```
GET http://localhost:8000/static/tailwind.min.js 404 (Not Found)
```
→ `download_tailwind.py`를 실행하여 Tailwind CSS 다운로드 필요

#### Network 탭

1. 페이지 새로고침 (F5)
2. 로드된 리소스 확인:

| 파일 | 상태 | 크기 |
|------|------|------|
| `localhost:8000/` | 200 | ~25KB |
| `tailwind.min.js` | 200 | ~150KB |
| `hana_logo.png` | 200 | ~12KB |

✓ 모든 파일이 **200 OK** 상태여야 함

## 통합 테스트 (전체 시스템)

전체 시스템을 테스트하려면 3개 서버가 모두 실행 중이어야 합니다:

```bash
# 터미널 1: TTS 서버
cd ../tts_server  # 또는 해당 디렉토리
python tts_server.py

# 터미널 2: 비디오 생성 서버
cd ../video_generation_server
python video_server.py

# 터미널 3: 웹 서버
cd ../web_demo
python app.py
```

### 전체 플로우 테스트

1. `http://localhost:8000` 접속
2. 텍스트 입력: "안녕하세요"
3. "비디오 생성하기" 클릭
4. 진행 상태 확인:
   - 음성 생성 중... (10% → 30% → 50%)
   - 비디오 생성 중... (50% → 70% → 85%)
   - 마무리 중... (85% → 95% → 100%)
5. 비디오 자동 재생
6. 다운로드 버튼 클릭하여 파일 저장 가능

## 트러블슈팅

### UI가 완전히 깨짐 (텍스트만 보임)

**원인**: Tailwind CSS가 로드되지 않음

**해결**:
1. F12 → Console 탭에서 에러 확인
2. `static/tailwind.min.js` 파일 존재 여부 확인
3. 파일이 없으면 `python download_tailwind.py` 실행
4. 서버 재시작

### 버튼을 클릭해도 반응 없음

**원인**: JavaScript 에러 또는 서버 연결 실패

**해결**:
1. F12 → Console 탭에서 JavaScript 에러 확인
2. F12 → Network 탭에서 API 요청 실패 여부 확인
3. 서버 터미널에서 에러 로그 확인

### 비디오 생성 실패

**원인**: TTS 또는 비디오 생성 서버 미실행

**해결**:
1. 서버 터미널에서 에러 메시지 확인
2. TTS 서버 (localhost:7009) 실행 확인
3. 비디오 생성 서버 (localhost:8001) 실행 확인
4. `/health` 엔드포인트로 서버 상태 확인:
   ```bash
   curl http://localhost:8000/health
   ```

## 성능 벤치마크

### 로딩 시간

- **초기 페이지 로드**: < 1초
- **Tailwind CSS 로드**: < 0.5초 (로컬 파일)
- **총 초기 로딩**: < 2초

### 리소스 사용

- **HTML**: ~25KB
- **Tailwind CSS**: ~150KB
- **로고 이미지**: ~12KB
- **총합**: ~190KB

→ **폐쇄망에서도 빠른 로딩 보장**
